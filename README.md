# Online School Management System

A web-based school management system built with HTML, CSS, and JavaScript. It provides separate dashboards for admins and students to manage school activities efficiently.

---

## Features

### Public Pages
- **Home** — School overview, teacher list, and upcoming events
- **Login / Signup** — User authentication pages

### Admin Dashboard
- **Student Management** — View and manage student records
- **Teacher Management** — Add, edit, and remove teachers
- **Notice Management** — Create, edit, and publish school notices
- **Event Management** — Add, edit, and manage school events
- **Admission Management** — Handle student admissions

### Student Dashboard
- **Main Dashboard** — Overview of student activities
- **Student Profile** — View and manage personal profile

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/IT22182678-Maleesha/Online_School_Management_System.git

2. Open the project folder:
   ```bash
   cd Online_School_Management_System
   ```

3. Open `index.html` in your browser (double-click or use a live server):
   - **VS Code**: Install "Live Server" extension → right-click `index.html` → "Open with Live Server"
   - **Direct**: Open `index.html` in any web browser
